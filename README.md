## 🧠 فكرة المشروع

**Smart Flashcards** هو تطبيق ويب تفاعلي يساعدك على حفظ المعلومات عبر نظام **التكرار المتباعد** (Spaced Repetition) داخل بيئة جذابة ومحفزة بصرياً.  
أجب عن الأسئلة، وسجّل تقدمك، وراجع أخطاءك، وأنشئ اختباراتك الخاصة – كل ذلك ضمن مجموعات (Decks) قابلة للتصدير والمشاركة.

> ⚡ **الميزة الأهم:** إذا أجبت خطأً، يُعاد السؤال إليك حتى تثبت فهمه. وإذا أجبت صحيحاً، تنتقل تلقائياً إلى التالي.

## 🖥️ العروض التوضيحية

| الميزة | الوصف |
| :--- | :--- |
| 📚 **مجموعات متعددة** | إنشاء، حذف، والتبديل بين مجموعات البطاقات. |
| 🎨 **خلفية ثلاثية الأبعاد** | مشهد Three.js ديناميكي (عقدة توروس + نجوم متحركة). |
| 🔊 **تأثيرات صوتية** | أصوات عند الإجابة الصحيحة والخاطئة (قابلة للكتم). |
| 📊 **إحصائيات شاملة** | عدد الصحيح/الخطأ، نسبة النجاح، وأصعب 3 أسئلة بناءً على الأخطاء. |
| 🔍 **مراجعة الأخطاء فقط** | تصفية الأسئلة التي أجبت عليها خطأً في الماضي. |
| 🧪 **اختبار مخصص** | اختيار أسئلة محددة (يدوي، عشوائي، تحديد الكل) وبدء اختبار مؤقت دون التأثير على المجموعة الأصلية. |
| 💾 **استيراد / تصدير JSON** | مشاركة الأسئلة مع الآخرين أو عمل نسخ احتياطي. |
| 🔗 **رابط مشاركة مشفر** | ضغط الأسئلة الحالية في رابط Base64 يمكن فكه في أي متصفح. |
| 📝 **شرح إضافي** | إضافة شرح لكل سؤال يظهر بعد الإجابة لتعزيز الفهم. |
| 📱 **تصميم متجاوب** | يعمل على الحاسوب والجهاز اللوحي والهاتف. |

## 🚀 كيفية التشغيل

### عبر الإنترنت (بدون تثبيت)
- فقط افتح ملف `index.html` في أي متصفح حديث (يفضل Chrome / Edge / Firefox).
- جميع البيانات تُحفظ تلقائياً في `localStorage` للمتصفح.
